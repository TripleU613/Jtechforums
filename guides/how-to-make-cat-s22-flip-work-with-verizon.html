How to Make a CAT S22 Flip Work with Verizon

This guide will walk you through each step, including how to install the necessary software and properly configure your phone. Be warned: this is an advanced process that may risk damaging your device. Proceed at your own risk! This guide is exported from Reddit.


Need help? Join our forums, where our amazing community members can assist you with any questions!

Requirements

  • Software:
    1. QPST v2.7.496.1 (Qualcomm Product Support Tools)
    2. Qualcomm USB Drivers
    3. Verizon MBN file (mcfg_sw.mbn)
  • Hardware:
    1. CAT S22 with stock Android 11 (must be network unlocked)
    2. Windows PC
    3. Data-capable USB-C cable

Step-by-Step Instructions

  • Install Qualcomm USB Drivers:
    1. Download and run setup.exe
    2. Reboot your computer after installation
  • Install QPST:
    1. Install QPST v2.7.496.1
    2. Verify installation of QPSTConfig.exe and PDC.exe in the QPST bin folder
  • Enable DIAG Mode on CAT S22:
    1. Open Phone app and dial ##3424#
    2. Tap “Open Port” on the DiagView screen
    3. Connect your phone to the PC using USB-C cable
  • Check Device Detection:
    1. Open Device Manager
    2. Look under “Modems” for “Qualcomm HS-USB Modem XXXX”
  • Configure in QPST:
    1. Open QPSTConfig.exe
    2. Go to “Ports” tab and ensure phone appears (or add manually)
  • Load Verizon MBN in PDC:
    1. Open PDC.exe
    2. Select your Qualcomm device from the dropdown
    3. Click “Load” and select mcfg_sw.mbn
  • Activate CDMAless-Verizon:
    1. Right-click CDMAless-Verizon → SetSelectedConfig → Sub0
    2. Click “Activate” once it says “pending”
    3. Ignore errors – verify it shows “Active”
  • Reboot and Verify:
    1. Restart your CAT S22
    2. Close the DIAG port
    3. Check that VoLTE is working

Optional Fixes

  • Set Preferred Network to LTE Only:
    1. Dial *#*#4636#*#*
    2. Select “Phone Information”
    3. Change preferred network to LTE only
    4. Toggle Airplane Mode on/off if needed
    5. Reboot if necessary
  • SMS Not Working?
    1. Call Verizon
    2. Ask them to configure your line as CDMAless
    3. More info: Reddit PSA Link
