How to Install Android Auto on a Cat S22 Flip

This guide teaches you how to run Android Auto on the Cat S22 Flip. Usually, it will not allow you to run it due to the phone running Android Go, but this fixes that issue.


Need help? Join our forums, where our amazing community members can assist you with any questions!

Requirements

  • Magisk must be installed on the device for this to work. See our forum for a guide on how to do that.
  • The Magisk Module File on the phone.

Step 1: Install the Android Auto Magisk Module

  1. Open the Magisk app.
  2. Tap Modules > Install from Storage.
  3. Select the Android Auto Magisk Module ZIP file.
  4. Wait for installation to complete, then tap Reboot.

Step 2: Enable Android Auto

  1. Update Android Auto via the Google Play Store by searching for it and pressing Update
  2. Go to Settings > Apps > See All Apps > Android Auto > Permissions
    3. and Allow all permissions
  3. Go to Settings > Apps > Advanced > Special App Access > Notification Access. Find Android Auto and toggle it ON.

Important: Before Removing The Module

You must uninstall Android Auto updates before removing the Magisk Module, or you may get stuck in a bootloop and need a Factory Data Reset. Search for Android Auto in the Google Play Store ad press Uninstall. Once updates are removed, you can safely uninstall the Magisk Module from the Magisk app under the Modules section.
